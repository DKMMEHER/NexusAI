# Contributing to NexusAI

Thank you for your interest in contributing to NexusAI! We welcome contributions from the community to help make this Generative AI suite even better.

## ü§ù How to Contribute

1.  **Fork the Repository**: Click the "Fork" button on the top right of the repository page.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/NexusAI.git
    cd NexusAI
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/AmazingFeature
    ```
4.  **Make Changes**: Implement your feature or fix.
5.  **Commit Changes**:
    ```bash
    git commit -m "Add some AmazingFeature"
    ```
6.  **Push to Branch**:
    ```bash
    git push origin feature/AmazingFeature
    ```
7.  **Open a Pull Request**: Go to the original repository and click "New Pull Request".
